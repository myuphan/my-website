Management: Task Tracker

Task Tracker is a project designed to track and manage tasks. In this project, we focus on building a command-line interface (CLI) application to practice core software development concepts such as CRUD operations, data storage, and user interaction. In real-world business, task tracker project can develop into employee task management system or customer support ticket system.

📄 JSON function

We use json.dump() and json.load() from Python's json module to convert data between Python objects and JSON files. json.dump() writes a python object to json file and json.load() reads data from json file and converts it back to python objects

Python

FILE_NAME = "tasks.json"
                                            
def load_tasks():
with open(FILE_NAME, "r") as file:
        return json.load(file) #Json converts the data into a python object

 
def save_tasks(tasks):
    with open(FILE_NAME, "w") as file:
        json.dump(tasks, file, indent=4)

👀 CRUD operation

In this section, we implement basic CRUD operations such as creating, reading, updating, and deleting tasks. In addition, users can update their task progress and filter tasks based on their status. We add two specific functions to handle progress updates and task filtering 

def create_task(description):
    tasks = load_tasks() #new local variable to load the tasks from the json file
    task = {
        "id": len(tasks) + 1,
        "description": description,
        "status": "to do"
    } #task formatted as a dictionary with id, description and status
    tasks.append(task)  #add to the tasks list
    save_tasks(tasks)
    print(f"Task '{description}' created successfully.") #Json load is updated with the new task and saved back to the file

def update_task():
    list_tasks()
    task_id = input("Enter the task ID to update: ")
    tasks = load_tasks()
    for task in tasks:
        if task["id"] == int(task_id):
            new_description = input("Enter the new description: ")
            task["description"] = new_description
            save_tasks(tasks)
            print(f"Task ID {task_id} updated successfully.")
            return #exit function after updating the task
    print(f"Task ID {task_id} not found.")
   

def delete_task():
    list_tasks() #display the list of tasks before deleting
    task_id = input("Enter the task ID to delete: ")
    tasks = load_tasks()
    for task in tasks:
        if task["id"] == int(task_id):
            tasks.remove(task)
            save_tasks(tasks)
            # tasks = reindex_tasks(tasks)
            print(f"Task ID {task_id} deleted successfully.")
            return #exit function after deleting the task
    print(f"Task ID {task_id} not found.")


def mark_status(task_id, status):
    list_tasks()
    tasks = load_tasks()    
    for task in tasks:
        if task["id"]==int(task_id): #inside vs outside data
            task["status"] = status #update the status of the task with the given task_id
            save_tasks(tasks)
            print(f"Task ID {task_id} marked as {status}.")
            return
    print(f"Task ID {task_id} not found.")

def list_tasks():
    tasks = load_tasks()
    if not tasks:
        print("No tasks found.")
        return
    for task in tasks:
        print(f"ID: {task['id']}, Description: {task['description']}, Status: {task['status']}")
    return

Note: for create function, after append a new task to tasks, the task is just added to python memories. We need to save it back to JSON file. Hence, ` save_tasks(tasks)` is neccessary. In addition, since we use the task ID to delete or update tasks, it is important to show users the list of tasks along with their corresponding IDs. This helps users identify which task and ID they want to select. 

def categorize_tasks():
    tasks = load_tasks()

    if not tasks:
        print("No tasks found.")
        return

    task_list = input("Filter by (to do, in progress, done): ")
    if task_list not in ["to do", "in progress", "done"]:
        print ("Invalid answer. Please enter: to do, inprogress or done")
    found = False
    for task in tasks:
        if task["status"] == task_list:
            found = True
            print(task)

    if not found: #found == false
        print(f"no matching activities with status '{task_list}' found")

For categorizing tasks, we want the function to return tasks based on the user's input. Since the JSON file stores data as a list of dictionaries ([] containing {}), where each dictionary has elements such as id, description, and status, we need to iterate through each task (dictionary) in the list (tasks). During each iteration, we check if the task's status task["status"] value matches the user's input. If it matches, we return/display that task.

To improve usability, we use Python's argparse module to create a command-line interface (CLI). This allows users to interact with the program through simple commands without needing to view or understand the underlying code. Users can execute actions such as creating, updating, deleting, or listing tasks directly from the terminal.

💻 Argparse

Learn about argparse 

def main():
    parser = argparse.ArgumentParser(description="task tracker input")
    subparsers = parser.add_subparsers(dest="command")

    create = subparsers.add_parser("create", help="create a new task")
    create.add_argument("description", help="task description") #function create have the description argument
    

    update = subparsers.add_parser("update", help="update task")

    delete = subparsers.add_parser("delete", help="delete task")

    mark_task = subparsers.add_parser("change", help= "mark task inprogress or complete")
    mark_task.add_argument(
    "task_id",
    type=int,
    help="task id")

    mark_task.add_argument(
    "status",
    help="new status")


    list_task = subparsers.add_parser("list", help="list all tasks")
    categorize_task = subparsers.add_parser("categorize", help="filter task by status")
    
    args = parser.parse_args()
    if args.command == "create":
        create_task(args.description)
    elif args.command == "update":
        update_task()
    elif args.command == "delete":
        delete_task()
    elif args.command == "change":
        mark_status(args.task_id, args.status)
    elif args.command == "list":
        list_tasks()
    elif args.command == "categorize":
        categorize_task()
    else:
        parser.print_help()
    
    


if __name__ == "__main__":
    main()
